La Dulce Monarca website files

What is included:
- index.html
- styles.css
- assets/ folder with all images

How to publish on Vercel:
1. Create a new GitHub repository.
2. Upload every file in this folder to the repository.
3. In Vercel, import the GitHub repository and deploy.
4. Add your custom domain: ladulcemonarca.com
5. Follow the DNS instructions Vercel gives you.

Notes:
- This is a static website, so it is simple and fast to deploy.
- The contact form currently opens the visitor's email app.
- The Instagram section is ready for a real embed later.
